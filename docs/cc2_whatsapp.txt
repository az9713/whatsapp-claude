                                                                                                  Claude Code Features                                                                          
  Skills (.claude/skills/ directory):
  - Markdown files with frontmatter that extend Claude's capabilities
  - Support custom slash commands, arguments, tool restrictions
  - Can be user-invocable or auto-triggered

  MCP (Model Context Protocol):
  - Connect Claude to external tools via servers (stdio, HTTP, SSE)
  - Popular servers: GitHub, Sentry, PostgreSQL, filesystem
  - Built-in Chrome integration available

  Hooks:
  - Trigger on events: PreToolUse, PostToolUse, Stop, SessionStart, etc.
  - Can run bash commands or prompt-based logic
  - Used for code formatting, notifications, validation

  Subagents:
  - Specialized agents (Explore, Plan, etc.)
  - Can create custom subagents with specific tools/permissions

  Remotion Fundamentals

  - Videos = React components that render frame-by-frame
  - useCurrentFrame() - get current frame number (0-indexed)
  - useVideoConfig() - get width, height, durationInFrames, fps
  - <Composition> - registers video with metadata in src/Root.tsx
  - Frame-based animation: change content per frame → video

  Chrome DevTools Protocol (CDP)

  - JSON-based messaging for browser instrumentation
  - 60+ domains: DOM, Network, Page, Input, Debugger, etc.
  - Connect via WebSocket (--remote-debugging-port) or HTTP
  - Use cases: automation, screenshots, DOM manipulation, network interception

      119 +- Connect via WebSocket to `--remote-debugging-port`

      124 +{
      125 +  "mcpServers": {

      129 +      "env": { "CDP_PORT": "9222" }
      130 +    }
      131 +  }
      132 +}
      133 +```
