     ## Project Overview

     WhatsApp-Claude bridges WhatsApp messaging with Claude Code CLI. Users send `/claude
     <task>` messages to themselves on WhatsApp, the bot spawns Claude Code to execute the
     task, and sends results back via WhatsApp.

     **Flow:** `WhatsApp → whatsapp-bot.js → claude CLI child process → response →
     WhatsApp`
